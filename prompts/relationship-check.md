# Prompt: Relationship Check

---

## Prompt

```
CoSy, run a relationship check.

Pull from:
- relationships/context/ -> all stakeholder files
- team/context/ -> all direct report files
- memory/interactions.md -> recent interaction log

Answer:

1. **Overdue touchpoints** -- Who haven't I connected with recently that I should?
   Flag anyone I haven't had a meaningful interaction with in 2+ weeks.

2. **Open commitments** -- Did I promise anything to someone that isn't done yet?
   Check all open threads across team and relationship files.

3. **Relationship health** -- Anyone where the dynamic feels off or needs attention?

4. **Quick wins** -- A message or short note that would strengthen a relationship
   right now with minimal time investment.

Keep it actionable. Give me names and specific suggested actions.
```
